# UBP Demo — Overview

This demo shows how UBP links can appear inside websites, applications, or wallets.

## Purpose
The purpose of this demo is to visually demonstrate how clean and simple the UBP link format is.  
It is not meant for validation or real parsing yet.

## Included in This Folder
- index.html — a minimal HTML page displaying sample UBP links

## Supported Example Formats
ubp:btc/tx/111aaa
ubp:eth/address/0xABCDEF123
ubp:sol/token/So111111111111111111

## How to Use
1. Download this repository
2. Open demo/index.html in any browser
3. You will see UBP links displayed as plain text
