# Contributing to UBP

Thank you for your interest in contributing to **UBP — Universal Blockchain Protocol**! We welcome all help, from documentation improvements to code libraries, to new chain integrations.

## How to Contribute

1. **Fork** this repository  
2. Create a branch: `git checkout -b my-feature`  
3. Make your changes  
4. Add tests / examples if relevant  
5. Submit a **Pull Request (PR)** with a clear description of what you changed

## Types of Contributions

- Documentation — Improve spec, fix grammar, add examples  
- Code / Integration — Parsers, resolvers, SDKs  
- Chain Support — Add new chain IDs, types, or rules  
- Web / UI — Demo improvements, website, converters

## Code Style

- Use **lowercase** for chain codes (e.g., `tri`, `eth`)  
- Use dash or underscore for branch names

## Communication

- Open **Issues** for bugs or feature requests  
- Use **Pull Requests** for proposed changes  
- Be respectful, detailed, and clear  

Thank you — you're helping build something that can standardize blockchain links for everyone!
